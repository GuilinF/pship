项目名称：基于Django+BootStrap的船舶信息管理系统。
项目功能：
1.航次信息的增删改查，船名联想输入。

2.燃油信息的增删改查。

3.货物信息的增删改查。

4.港口信息的增删改查。

5.总预算的自动计算。
